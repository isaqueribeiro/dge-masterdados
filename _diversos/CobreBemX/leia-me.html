CobreBemX

As soluções de meios de pagamento Cobre Bem são sinônimo de confiabilidade, eficiência e segurança. Sendo assim, a Cobre Bem criou o selo "Meio de Pagamento Cobre Bem" para empresas que desejam claramente informar ao seu cliente que são usuários desta marca, que hoje lidera o mercado de meios de pagamento no Brasil.
A Cobre Bem sente-se muito honrrada em tê-la como cliente e deseja-lhe boas cobranças.
Se desejar exibir este selo em seu site, favor criar um link para www.cobrebem.com.

Instalação:

Para instalar o CobreBemX no seu computador, siga o procedimento abaixo:
  1. Descompacte o arquivo de instalação em um diretório desejado;
  2. Execute o arquivo "Instalar.exe" para registrar a DDL no seu Windows.
Para instalar o CobreBemX no computador do seu cliente em conjunto com o programa desenvolvido à parte, siga o procedimento abaixo:
  1. Copie o arquivo CobreBemX.dll para qualquer diretório do computador do cliente. De preferência que seja o mesmo diretório onde o programa está instalado;
  2. Registre a DLL no Windows utilizando, por exemplo, o arquivo ''Instalar.exe'' contido na InstalShield, basta incluir o arquivo ''CobreBemX.dll'' em sua aplicação e solicitar que o mesmo seja registrado no Windows pelo instalador;
  3. Inclua também os arquivos de imagem que o cliente irá necessitar para utilizar as funções de Envio de Boleto por E-Mail, Geração de HTML de Boleto e Impressão de Boletos. Sugerimos que todos sejam instalados.

Exemplos para utilização do componente:

Acessando a URL a seguir "ExemplosUtilizacao.html",você será redirecionado para a página de downloads do nosso site. Nesta você encontrará os fontes dos exemplos fornecidos pela Cobre Bem que demonstram a utilização das funcionalidades do CobreBemX, lembramos que os fontes são de exemplo, podendo ser alterado de acordo com sua necessidade.

Importante: Os exemplos que acompanham o componente utilizam banco de dados Paradox (Delphi) e Microsoft Access (Visual Basic), porém você pode utilizar o banco de dados que desejar para armazenar os seus dados.

Chaves de Licenças para realização de Testes:

No diretório "Exemplos/Licenças", há apenas o arquivo "arquivo de licença para teste", este utilizado para realizar quaisquer tipos de testes que julgar necessário, tendo disponível todas as funcionalidades que o componente oferece.

Para efetuar os seus testes com o CobreBemX em qualquer um dos bancos suportados, siga as instruções contidas na página Características Bancos e Carteiras.

Chaves de Licenças - Definitivas

Ao adquirir a Licença de Uso do Componente CobreBemX serão encaminhadas as chaves para gerar os arquivos de licenças para os bancos e carteiras do seu Cedente.

O arquivo de licença é gerado para cada Cedente/Banco/Carteira e portanto não serve para qualquer banco, somente para o que foi gerado.

Uma Chave de Licença Definitiva permite a geração para um mesmo Cedente, identificado pelo CPF ou CNPJ de quantos arquivos de licença forem necessários para diversos Bancos/Carteiras.

Os arquivos de licença podem ser gerados através da URL: http://licenca.cobrebemx.com.br

Ao preencher as informações necessárias e solicitar a geração do arquivo de licença, automaticamente será iniciado o download do arquivo, exceto para os navegadores IE 5.5, que será necessário exibir o código fonte e salvar manualmente o arquivo gerado.

Documentação Técnica

A Documentação Técnica do CobreBemX está composta por todos os Objetos, Propriedades e Métodos existentes no componente, incluindo suas respectivas definições.

Para que a Documentação Técnica do CobreBemX seja compreendida mais facilmente é interessante visualizar a Hierarquia de Objetos.

A Hierarquia de Objetos, funciona, como uma ''árvore'', onde será possível visualizar de maneira rápida e fácil as propriedades e objetos existentes no CobreBemX.

Para maiores detalhes, utilize os exemplos práticos disponibilizados no site da Cobre Bem Tecnologia, a partir de Downloads CobreBemX.

A documentação do CobreBemX pode ser visualizada através do arquivo ''Documentacao\CobreBemX.htm''. Você terá acesso a mesma, após a realização do download deste componente. Neste arquivo contém a lista dos objetos suportados, seguidos de suas propriedades e métodos.

Para orientá-lo melhor, disponibilizamos a Documentação Técnica, veja: Documentação Técnica CobreBemX.

Informação ao Desenvolvedor

Sugerimos a leitura do Manual de Noções Básicas de Cobrança e principalmente aconselhamos que seja orientado ao cliente, futuro cedente a leitura do item ''Orientação para o Cedente'' existente no referido texto.

O Manual de Noções Básicas de Cobrança está disponível também em PDF na área de suporte do site da Cobre Bem Tecnologia.

Desinstalação

Para retirar o CobreBemX do seu computador, basta removê-lo do seu computador, através da execução do arquivo ''Desinstalar.exe''.

Para maiores informações, acesse o Manual Técnico disponível no site.
